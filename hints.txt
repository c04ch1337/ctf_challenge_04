# Hints for CTF Challenge 4

1. **Flag 1**: Exploit a SQL injection vulnerability in the login page.
2. **Flag 2**: Find a hidden directory using directory brute-forcing.
3. **Flag 3**: Exploit a file upload vulnerability in the upload feature.
4. **Flag 4**: Decode a base64-encoded string in the page source.
5. **Flag 5**: Brute-force the SSH service to find weak credentials.
6. **Flag 6**: Exploit a weak SSL/TLS configuration to intercept traffic.
7. **Flag 7**: Exploit a command injection vulnerability in the admin panel.
8. **Flag 8**: Find a flag hidden in the response headers.
9. **Flag 9**: Exploit a server-side template injection (SSTI) vulnerability.
10. **Flag 10**: Decode a JWT token to find the flag.
